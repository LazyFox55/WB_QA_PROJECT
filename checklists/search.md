# Чек-лист: Поиск товара — Wildberries.ru

## 1. Позитивные сценарии
- [ ] Поиск существующего товара (по полному названию)
- [ ] Поиск существующего товара (по части названия)
- [ ] Поиск товара с использованием фильтров (цена, бренд, размер)
- [ ] Сортировка результатов выдачи (по цене, по популярности)
- [ ] Нажатие на "Очистить" в строке поиска
- [ ] Переход на страницу карточки товара из результатов поиска

## 2. Негативные сценарии
- [ ] Поиск несуществующего товара (проверка сообщения "Ничего не найдено")
- [ ] Поиск товара с использованием спецсимволов (!@#$%)
- [ ] Поиск товара с использованием пустой строки (проверка валидации)
- [ ] Поиск товара с использованием цифр в названии (если применимо)

## 3. UI/UX
- [ ] Отображение превью-картинок товаров
- [ ] Наличие цены и рейтинга у товара
- [ ] Адаптивность поисковой строки при изменении ширины окна браузера
