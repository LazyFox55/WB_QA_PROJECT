# Чек-лист: Авторизация — Wildberries.ru

## 1. Регистрация / Вход
- [ ] Вход по корректному номеру телефона
- [ ] Получение SMS-кода для подтверждения
- [ ] Ввод корректного SMS-кода
- [ ] Повторная отправка SMS-кода (таймер)

## 2. Негативные сценарии
- [ ] Ввод некорректного номера телефона (меньше 10 цифр)
- [ ] Ввод неверного SMS-кода
- [ ] Попытка войти без номера телефона (проверка валидации кнопки)
- [ ] Использование просроченного SMS-кода

## 3. UI/UX
- [ ] Отображение маски номера телефона при вводе
- [ ] Наличие кнопки "Назад" или отмены
- [ ] Валидация полей в реальном времени (подсветка ошибок)
