## 1. Введение

Данный документ описывает план тестирования веб-приложения Wildberries.ru.  
Проект выполнен как учебный прект для демонстрации навыков ручного тестирования.

**Тестируемый объект:** https://www.wildberries.ru  
**Автор:** Жданова Милана  
**Дата завершения:** 15 июня 2026  

---

## 2. Цели тестирования

- Проверить корректность работы ключевых пользовательских сценариев
- Выявить дефекты в функциональности поиска, корзины и авторизации
- Задокументировать найденные баги и составить тест-кейсы для регрессии

---

## 3. Объект тестирования

| Модуль | Описание |
|---|---|
| Поиск | Строка поиска, фильтры, результаты выдачи |
| Корзина | Добавление товаров, изменение количества, удаление |
| Авторизация | Регистрация, вход по номеру телефона |

---

## 4. Виды тестирования

| Вид | Описание |
|---|---|
| Функциональное | Проверка соответствия функций требованиям |
| Негативное | Проверка поведения при некорректных входных данных |
| UI/UX | Проверка отображения элементов и удобства использования |
| Кросс-браузерное | Chrome, Firefox, Yandex Browser |
| Мобильное | Проверка на мобильном устройстве (Android) |

---

## 5. Что не входит в scope

- Нагрузочное тестирование
- Тестирование оплаты (требуются реальные данные карты)
- Тестирование доставки и пунктов выдачи
- Тестирование мобильного приложения

---

## 6. Окружение

| Параметр | Значение |
|---|---|
| ОС | Windows 11 |
| Браузер | Chrome (актуальная версия) |
| Разрешение | 1920x1080 |
| Подключение | Wi-Fi |

---

## 7. Критерии начала тестирования

- Сайт доступен и загружается без ошибок
- Основные разделы открываются корректно (smoke пройден)

---

## 8. Критерии окончания тестирования

- Все тест-кейсы выполнены
- Найденные баги задокументированы в баг-репортах
- Критических блокеров нет или они задокументированы

---

## 9. Риски

| Риск | Влияние | Митигация |
|---|---|---|
| Изменение UI сайта | Тест-кейсы устареют | Обновлять тест-кейсы при изменениях |
| Нестабильная работа сайта | Ложные баги | Воспроизводить баг минимум 2 раза |
| Нет доступа к БД | Невозможно проверить данные через SQL | Использовать публичные тестовые API |
